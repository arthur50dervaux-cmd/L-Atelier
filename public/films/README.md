# Films & rendus cinématographiques

Déposez ici vos vidéos de rendus (haute définition).

- Format recommandé : **`.mp4`** (codec H.264 ou H.265), 1080p ou 4K
- Conseil : compressez avec [HandBrake](https://handbrake.fr) pour un poids raisonnable (préréglage « Web » / « Vimeo »)

## Fichiers attendus par le site
| Emplacement dans le site | Nom de fichier attendu |
|---|---|
| Showreel principal | `showreel.mp4` |
| Villa Horizon | `villa-horizon.mp4` |
| Maison des Vignes | `maison-vignes.mp4` |
| Calanques | `calanques.mp4` |

Vous pouvez changer ces noms via les attributs `data-film` dans
`index.html`. Une image d'aperçu (poster) peut être ajoutée via
`data-poster="films/mon-poster.jpg"`.

Tant qu'un fichier est absent, un message d'aide s'affiche à la place
de la vidéo.

> Astuce : pour des films lourds (4K), héberger sur **Vimeo/YouTube** et
> intégrer l'iframe est souvent préférable au fichier `.mp4` brut. Demandez
> et je peux brancher un lecteur Vimeo/YouTube à la place.
