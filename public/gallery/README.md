# Images HD des projets

Déposez ici les photos / rendus haute résolution de vos projets.

- Formats : `.jpg`, `.webp` (recommandé pour le poids), `.png`
- Résolution conseillée : largeur 2000–2560 px

## Comment relier une image à un projet
Dans `index.html`, chaque carte de projet possède un attribut
`data-image`. Renseignez-y le chemin de votre image, par exemple :

```html
<article class="project-card" data-image="gallery/villa-belvedere.jpg" ...>
```

L'image remplacera alors le dégradé de démonstration, sur la carte **et**
dans la fenêtre de détail du projet.
